# Portfolio Optimizer App

A modern, premium web application for portfolio management and optimization. This app allows users to analyze their stock portfolios, visualize allocations, and optimize weights based on financial metrics like Sharpe Ratio and Omega Ratio.

## Features

- **Asset Management**: Add stocks to your portfolio with real-time color coding.
- **Portfolio Allocation**: Adjust weights and visualize your portfolio composition.
- **Financial Metrics**: Calculate Expected Return, Volatility, Sharpe Ratio, and Omega Ratio.
- **Optimization**: Run Monte Carlo simulations to find the efficient frontier and optimal portfolio weights.
- **Premium UI**: Glassmorphism design with a responsive and interactive interface.

## Tech Stack

- **Frontend**: React (Vite)
- **Styling**: TailwindCSS
- **Charts**: Recharts
- **Math**: Math.js

## Getting Started

1.  **Install Dependencies**:
    ```bash
    npm install
    ```

2.  **Run Development Server**:
    ```bash
    npm run dev
    ```

3.  **Build for Production**:
    ```bash
    npm run build
    ```
